# ДЗ21: Многопоточность в C++

Консольное приложение, моделирующее обработку задач N потоками

## Сборка

```bash
mkdir build
cd build
cmake ..
cmake --build .
.\Debug\main.exe
```

## Что делает программа

- создает очередь задач
- добавляет задачи с номерами 1-20
- запускает N рабочих потоков
- каждый поток берет задачу из очереди и обрабатывает ее 1 секунду
- после завершения всех задач потоки завершаются корректно
